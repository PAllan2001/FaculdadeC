# 📚 Algoritmos Reescritos em C

Este repositório reúne exercícios e pequenos algoritmos que eu originalmente desenvolvi em **Java** e posteriormente **reescrevi na linguagem C**.

A ideia surgiu quando comecei a estudar **C na faculdade** e decidi utilizar os mesmos problemas e desafios que já havia resolvido anteriormente para aprofundar meu entendimento da nova linguagem.

Ao reimplementar esses algoritmos em C, pude comparar diferenças importantes entre as linguagens, principalmente em relação a:

* Sintaxe
* Manipulação de memória
* Entrada e saída de dados
* Estruturas de controle
* Organização do código

---

## 🎯 Objetivo do Repositório

O principal objetivo deste repositório é **reforçar a lógica de programação** e **consolidar o aprendizado da linguagem C**, utilizando problemas que eu já havia resolvido anteriormente em Java.

Esse processo ajuda a:

* Fixar melhor os conceitos da linguagem C
* Entender diferenças entre **linguagens de alto nível (Java)** e **linguagens mais próximas do hardware (C)**
* Praticar reestruturação e adaptação de algoritmos

---

## 📂 Estrutura do Repositório

Cada arquivo contém um pequeno algoritmo ou exercício específico.

Exemplos de temas presentes nos códigos:

* Estruturas condicionais
* Entrada de dados pelo terminal
* Simulações simples
* Exercícios de lógica de programação

---

## 🛠️ Tecnologias Utilizadas

* **C**
* **Java** (versão original dos algoritmos)

---

## 📖 Motivação

Reescrever algoritmos em outra linguagem é uma ótima forma de aprendizado, pois obriga o programador a entender **a lógica por trás da solução**, e não apenas a sintaxe da linguagem utilizada.

Este repositório faz parte do meu processo de evolução nos estudos em **programação e ciência da computação**.
